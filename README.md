# AI Trading Bot

A full-stack AI-powered trading bot that analyzes market trends and provides automated trading insights.  
Built with a modern React frontend, Supabase backend, and interactive charts, this project combines AI, real-time data, and user-friendly design.

---

## Features

- Real-time market data and trading signals  
- Interactive charts with technical indicators  
- User authentication and portfolio management via Supabase  
- Forms with validation for adding strategies and settings  
- Modular design for extending AI models or trading strategies  

---

## Tech Stack

### Frontend
- React 18 with TypeScript  
- Vite (build tool)  
- Tailwind CSS with custom design tokens  
- shadcn/ui (UI components)  
- Lucide React (icons)  

### Data & State Management
- TanStack Query (server state)  
- React Router (navigation)  

### Backend
- Supabase (PostgreSQL, authentication, real-time)  

### Charts
- Recharts for trading charts and technical indicators  

### Forms & Validation
- React Hook Form  
- Zod (schema validation)
